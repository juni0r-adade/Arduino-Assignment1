# Arduino Assignment 1 – Smart Countdown

This is my submission for Assignment 1 in Programming C++ for Engineers Using Arduino.

## What the program does

The program counts down from 5 to 1 using a while loop.
For each number:
- The count is printed on the Serial Monitor.
- The onboard LED blinks that many times.

After the countdown, the message
"=== Countdown Complete ==="
is displayed.

## Concepts Used
- Variables
- Functions
- While Loops
- Serial Monitor
- LED Control
